# C# OOP Course Project

## Description
This project was developed as a final assignment in an Object-Oriented Programming (OOP) course using C#.

The system models courses and students using nested classes and includes LINQ-based queries.

## Features
- Course class with nested Student class
- Student grade management using List
- Query system based on user input:
  - Passing / failing students
  - C# courses filtering
  - Students with perfect scores
- LINQ queries (Lambda expressions)
- No constructors – uses properties only

## Technologies
- C#
- .NET Core
- LINQ

## Notes
- The project follows strict OOP principles
- Queries are implemented without loops
- Focus on clean structure and generic programming
