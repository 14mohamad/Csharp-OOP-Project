# 💻 C# OOP Course Project

## 📌 Description
This project was developed as a final assignment in an Object-Oriented Programming (OOP) course using C#.  
It demonstrates core OOP principles such as classes, encapsulation, and program logic implementation.

---

## 🛠️ Technologies
- C#
- .NET Core
- Visual Studio / VS Code

---

## 📂 Project Structure
- `Matala.sln` – Solution file  
- `Matala/` – Main project folder  
  - `Program.cs` – Entry point of the application  
  - `Course.cs` – Class implementation and core logic  

---

## ⚙️ Features
- Object-Oriented design using classes  
- Separation of logic into different components  
- Simple and clear structure for readability  
- Demonstrates basic C# programming concepts  

---

## ▶️ How to Run
1. Open the project in Visual Studio or VS Code  
2. Build the solution  
3. Run the program  

---

## 🎯 Purpose
The purpose of this project is to practice and demonstrate understanding of:
- Object-Oriented Programming (OOP)
- Class design and structure
- Writing clean and organized C# code
